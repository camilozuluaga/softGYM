Acciones

En este apartado encontraremos varios submenus donde podemos hacer las sigientes acciones

REGISTRAR ENTRADA AUTOMATICA.

CREAR SOCIO.

CREAR MEMBRESIA.

CREAR USUARIO DEL SISTEMA.

CIERRE DE CAJA.


Registrar entrada automatica.

Aqui vamos a registrar una entrada de un socio que ya este registrado en el sistema y que tenga una membresia activa vigente.


En este campo podemos digitar el codigo del socio ya sea con el teclado o con el huellero,despues de esto abrira la puerta si es que tiene una membresia activa; sino, este le avisara que no puede entrar


Crear socio del sistema.

Aqui vamos vamos a crear un usuario del sistema o empleado, donde le podemos dar acceso a algunas ventajas que cuenta este sistema.


Llenamos los campos necesarios para poder registrar un empleado o un administrador del sistema, el administrdor del sistema tiene acceso a todas las ventanas y puede editar y crear algunas membresias, socios o hasta puede manejar todas la contabilidad de la aplicacion, encambio el empleado solo puede accese a las ventanas o a las acciones que el adimistrador le de el permiso de acceder, para ello en la tabla inferior chulearemos a los permiso que le daremos al nuevo empleado.

La clave que se genera automaticamente es la primera letra del apellido junto con la identificacion suministrada al momento de crear un nuevo usuario.


Crear Socio, crear membresia y cierre de caja; estos temas se encuentran en otras unidades, podemos darle click en los menus que estan al lado iquierdo de esta submenu.
